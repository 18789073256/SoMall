# SoMall

**一个社交( Socail)+商城(Mall) 的开源学习项目**

#### 技术栈:

.netcore 3.1
vue
uni-app
typescript
docker
mssql

#### 前端

vue-element
uni-app

#### 微服务

Abp.vnext
RabbitMQ (Event Bus)
Ocelot (Api Gate)
Consul (Service Discovery)
Quartz (background Job Schedule)
