# SoMall

**一个社交( Socail)+商城(Mall) 的开源学习项目**

#### 技术栈:

.netcore 2.2
angular
typescript
docker
mssql(base on docker)

#### 前端

ng-zorro+ng-alain (Admin backEnd)
ionic (H5 and APP)
mpvue (weixin mini programe)

#### 微服务

Abp.vnext
RabbitMQ (Event Bus)
Ocelot (Api Gate)
Consul (Service Discovery)
Quartz (background Job Schedule)
